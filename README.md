# 2048 game with AngularJS

You can play the game online at

> https://dominguesgabe-2048.netlify.app/

## Setup Steps

### Clone Repository

```
git clone https://github.com/dominguesgabe/2048.git
```

### Npm install dependencies

```shell
$ npm install
```

### Run tests

```shell
$ npm run test
```
Run tests with coverage
```shell
$ npm run coverage
```
